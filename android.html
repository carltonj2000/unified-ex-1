Use scrcpy for mirroring android phone on windows/linux/mac. The requires that ADB is up and running. On Linux run ADB as root to save yourself a whole bunch of headaches. If you run into issues unplug the USB, Revoke USB debugging authorizations. Plug in the phone again and accept the on screen authorization. You can then log out of root to run adb or scrcpy. When adb devices is NOT run as root the adb output shows unauthorized as root you see device and now adb shell works as root or a regular users.
